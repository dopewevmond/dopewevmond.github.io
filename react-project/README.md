# Chuck Norris Jokes

This was built with React and Bootstrap CDN, and the [Chuck Norris API](https://api.chucknorris.io/). 

It allows users to get either random jokes or jokes by category from the Chuck Norris API.

All React code can be found [here](https://github.com/dopewevmond/dopewevmond.github.io/blob/main/react-project/script.js).

Live demo [here](http://dopewevmond.me/react-project/index.html).
